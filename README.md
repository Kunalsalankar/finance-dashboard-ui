# Finance Dashboard (React + Tailwind)

Modern, clean, responsive finance dashboard using mock frontend data only.

## Features Implemented

- [x] Dashboard Overview with summary cards
- [x] Time-based visualization (Balance Trend line chart)
- [x] Categorical visualization (Spending Breakdown pie chart)
- [x] Transaction list with details (date, amount, category, type)
- [x] Transaction filtering (All / Income / Expense)
- [x] Transaction sorting and search
- [x] Role-based UI (Viewer and Admin)
- [x] Insights section
- [x] State management with React Context
- [x] Responsive design (mobile/tablet/desktop)
- [x] Empty states for transactions/charts
- [x] Dark mode toggle (optional enhancement)
- [x] localStorage persistence (optional enhancement)

## Tech Stack

- React (functional components)
- Tailwind CSS
- Recharts
- React Router
